This Privacy Policy describes what data is collected and how it is used by the memory training application (hereinafter referred to as the “Application”).

1. General information about the Application

The Application is designed to develop memory and intellectual abilities. It helps users train their brain through exercises involving the addition of numbers and filling in missing values, as well as tracking personal progress.

The Application does not require registration, does not use network services, and does not transmit data to external servers.

2. What data is collected

The Application may store the following data exclusively locally on the user’s device:

  • user name or nickname (at the user’s discretion);
  • exercise progress, including:
    • task completion time;
    • number of mistakes made.

This data is used solely to display progress within the Application.

3. Data storage and processing

  • All data is stored only on the user’s device.
  • The Application does not transmit data over the internet.
  • The Application does not use remote servers, cloud storage, or third-party analytics services.

4. Data sharing with third parties

The Application does not share any user data with third parties, including advertising, analytics, or other services.

5. Access to device data

The Application does not request access to contacts, camera, microphone, location, or other sensitive device data.

6. Data security

Since all data is stored locally and not transmitted over the network, the risk of data leakage is minimal. The user is solely responsible for the security of their device.

7. Data deletion

All Application data can be completely deleted by the user by:

  • clearing the Application data in the device settings;
  • uninstalling the Application from the device.

8. Children’s data

The Application does not collect personal data and may be used by users of any age. It does not contain content that violates Google Play requirements for apps intended for children.

9. Changes to this Privacy Policy

The developer reserves the right to update this Privacy Policy. The current version should always be available to the user together with the Application.

10. Contact information

If you have any questions regarding this Privacy Policy, you may contact the developer using the contact details provided on Google Play.
